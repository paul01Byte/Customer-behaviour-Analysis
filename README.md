## 📌 Overview

This project focuses on analyzing customer purchasing behavior in a retail/e-commerce setting to generate **data-driven business insights**.

Instead of just analyzing data, the project answers **real-world business questions** like:
- How can we increase customer retention?
- Do discounts actually improve revenue?
- Who are the most valuable customers?

The end goal is to **bridge the gap between data and decision-making**.

---

## 🎯 Business Problem

Retail companies often face:
- Low customer retention  
- Ineffective marketing campaigns  
- Poor understanding of customer segments  

This project solves these by analyzing:
- Purchase patterns  
- Customer loyalty  
- Discount impact  
- Product performance

## 🧹 Data Cleaning & Preprocessing

Real-world data is messy—this project demonstrates **industry-level data cleaning**:

- Handled missing values using **domain-based logic**
- Fixed category inconsistencies
- Removed duplicate records
- Standardized column names for SQL compatibility
- Applied **advanced imputation techniques**

## 📊 Exploratory Data Analysis (EDA)
